# Промпт для передачи работ другому агенту (xyliganimbot)

## ОЧЕНЬ ВАЖНО
При работе с проектом строго следуйте правилам из `.cursor/rules/conventions.mdc` и `.cursor/rules/workflow.mdc`. Все изменения должны быть согласованы перед реализацией.

## Ключевые документы

**Обязательно к прочтению перед началом работы:**
- [`docs/vision.md`](docs/vision.md) — техническое видение проекта, архитектура, принципы разработки
- [`docs/conventions.md`](docs/conventions.md) — правила разработки, стандарты кода, процесс работы
- [`docs/tasklist_v0.1.md`](docs/tasklist_v0.1.md) — план итераций и задач, текущий прогресс
- [`.cursor/rules/conventions.mdc`](.cursor/rules/conventions.mdc) — обязательные правила разработки
- [`.cursor/rules/workflow.mdc`](.cursor/rules/workflow.mdc) — процесс работы над итерациями
- [`docs/BACKLOG.md`](docs/BACKLOG.md) — бэклог
