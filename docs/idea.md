# xyliganimbot — рабочий Telegram-бот

## Цель
- Создать Telegram-бота для рабочего чата, который поможет с поиском ответом в небольшой базе знаний
- Сделать лёгкую архитектуру, пригодную для запуска дома, на VPS или в контейнере.

## Пользователи и контекст
- Работники одной компании, общающиеся в одном групповом чате Telegram.
- Общение в естественном языке; бот отвечает прямо в чате или «в ответ» на сообщения, чтобы не захламлять ленту.

## Функциональные требования
- Обработка команд и естественного языка через Telegram Bot API.
- Поиск ответов в базе знаний (документ в Google Docs, доступный по прямой ссылке).
- Простой журнал действий.

## Нефункциональные требования
- **Безопасность**: белый список ID чатов/пользователей, секретный токен, защита админских команд.
- **Удобство**: единообразные команды, понятные подтверждения, возможность «отменить действие» в течение короткого времени.
- **Расширяемость**: модульная архитектура, позволяющая добавлять новые сценарии.
