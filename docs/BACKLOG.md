# Бэклог проекта xyliganimbot

## Высокий приоритет (MVP)

### 1. Настройка проекта
- Структура папок
- Python окружение
- Базовые файлы конфигурации

### 2. Telegram интеграция
- Подключение к Bot API
- Long polling
- Проверка белого списка

### 3. Конфигурация
- Загрузка config.yaml
- Переменные окружения
- Валидация

### 4. Google Docs импорт
- Получение документа (HTML-экспорт в ZIP-архиве)
- Распаковка ZIP-архива
- Извлечение изображений из архива
- Сохранение HTML/Markdown в файл
- Сохранение изображений в папку data/images/

### 5. Поисковая система (семантический поиск)
- Разделение документа на разделы (по заголовкам из JSON)
- Векторизация разделов через embedding-модели (sentence-transformers)
- Семантический поиск через косинусное сходство
- Ранжирование результатов по score
- Возврат цитат (snippet или полный текст раздела)
- Ограничение результатов (топ-N)

### 6. Обработка команд
- /help
- Текстовые запросы
- Админские команды

### 7. Логирование и аудит
- Файловое логирование
- Ротация логов
- Журнал действий

### 8. Docker и Kubernetes
- Dockerfile
- K8s манифесты
- Инструкции по деплою
